
# NutriCare

A Django-based nutrition and health platform connecting customers with certified dieticians.

## Features
- **Multi-role system**: Admin, Customer, Dietician dashboards
- **Diet Plans**: Free and paid diet plans with step-by-step tracking
- **Custom Diet Plans**: Personalized diet plans based on customer requests
- **Workout Plans**: Free and paid workout plans with video support
- **Health Metrics**: BMI and BMR calculators with history tracking
- **AI ChatBot**: NLP-powered nutrition assistant (TensorFlow/Keras)
- **In-App Messaging**: Customer ↔ Dietician chat
- **Feedback & Ratings**: Diet plan and dietician rating system
- **PDF Uploads**: Dieticians can upload and share PDF diet plans

## Setup
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
